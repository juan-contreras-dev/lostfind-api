LostFind API

REST API design project to manage lost and found items within an educational environment.

Features
- Register lost or found items
- Retrieve all items or a specific item by ID
- Update item status
- JSON-based data structure
  
Technologies
- JavaScript
- Fetch API
- JSON
- REST API concepts

Endpoints
- GET /objetos
- GET /objetos/:id
- POST /objetos
- PUT /objetos/:id

Project Structure
- docs/LostFindAPI.docx – Full project documentation
  
Status
Conceptual project — currently being extended into a working API using Node.js and Express.
